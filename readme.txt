== Description ==

Upload zip file that contains plain text(.txt format files), and then let TXTAsPost process and schedule them.

Warning: the first line of .txt file must be TITLE and the rest will be treated as CONTAIN.

Supported features:

* Specify category/categories to post
* Specify tags to post
* Time Interval by hours
* Published time has two options: 
   1, Accroding to the time of latest post e.g. your existing latest post was published at 8:00, so next post will be published at 8:00 + time interval
   2, From now on e.g. first post publish immediately, next post will be published after time interval
* Error Log
* Support page.

TXTAsPost is still beta, feel free to report bug on <a href="http://www.change-my-life.com/txt-as-post/">Change My Life</a>

== Installation ==

1. Extract txtaspost.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "TXT As Post" In Setting
4. Follow the on-screen instructions
